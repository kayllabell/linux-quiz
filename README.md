### 💡 What is this?

An interactive, terminal-themed quiz I built to study for my Linux midterm at Algonquin College. Instead of using flashcards, I built something I could actually interact with — and writing the questions myself forced me to actually learn the material.

🟢 **[Click here to try it live →](https://kayllabell.github.io/linux-quiz/)**

---

### 📋 What it covers

| Topic | Examples |
|---|---|
| **Core commands** | `chmod`, `chown`, `grep`, `awk`, `cat`, `echo`, `ls` |
| **Job scheduling** | `cron`, `crontab`, `at`, `atq`, `atrm`, `atd` |
| **User management** | `useradd`, `passwd`, `chage`, `visudo` |
| **Networking** | `dig`, `ip addr`, `ssh`, `sftp` |
| **Filesystem** | `/home`, `/boot`, `/mnt`, `/etc` |
| **Compression** | `gzip`, `gunzip`, `bzip2`, `bunzip2` |

---

### ⚙️ How it works

- Boots with an animated terminal log sequence
- Steps through 40+ multiple choice and true/false questions
- Shows an explanation immediately after each answer
- Displays your final score at the end
- No install, no dependencies — runs entirely in the browser
